What is malarm?


malarm is an Android application which plays music in playlist at sleep time and wakeup time.

How to setup

  1. Copy some music files (mp3, m4a, m4v, mp4) into "music" directory of SD memory which is supported by your android device
  2. Create 2 playlists in M3U format, sleep.m3u and wakeup.m3u. sleep.m3u is sleep time playlist and wakeup.m3u is wakeup time playlist. An M3U playlist file is a text file which lists one music file per one line. Copy them into "music" directory of SD memory.
  3. Insert SD memory into your Android device.

Example of playlist

Help.m4a
ABC.m4a
11 Crazy about you.mp3

Example of SD memory

music
  |
  +-- sleep.m3u
      wakeup.m3u
      Help.m4a
      ABC.m4a
      11 Crazy about you.mp3

How to set alarm

Main screen

  1. Set time picker to wakeup time.
  2. Click "Set Alarm" button.
  3. Malarm application plays sleep music listed in sleep.m3u one hour. It plays wakeup music listed in wakeup.m3u at specified time by time picker.

How to stop or cancel alarm

Click "Cancel Alarm" button
